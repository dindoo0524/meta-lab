# Target User — 성장톡

## 주요 사용자

하루를 의미 있게 마무리하고 싶은 직장인/프리랜서

### 특징
- 20~40대
- 바쁜 일상에서 성찰할 시간이 부족
- 일기를 쓰고 싶지만 꾸준히 못 함
- AI에 관심 있지만 범용 챗봇은 산만하다고 느낌
- 10분 이하의 가벼운 루틴을 원함

### 니즈
- 하루를 간단히 정리하고 싶다
- 막연한 기분을 구체적으로 인식하고 싶다
- 내일 뭘 해야 할지 하나만 정하고 싶다
- 꾸준히 할 수 있는 짧은 루틴이 필요하다

### 사용 맥락
- 퇴근 후 혹은 잠자기 전
- 모바일로 짧게 (5~10분)
- 매일 또는 격일

## 비사용자 (Non-Target)

- 전문 상담/치료가 필요한 사람
- 생산성 도구를 찾는 사람
- AI와 긴 대화를 즐기는 사람
- 데이터 분석/통계를 원하는 사람
