# MVP Scope — 성장톡 v0

## 포함 (In Scope)

### 페이지
- `/` — 랜딩/인트로
- `/session` — 성장톡 세션
- `/result` — 결과/요약

### 기능
- 하루 느낌 자유 입력
- 3개 구조화된 질문 순차 표시
- 각 질문에 대한 답변 입력
- 짧은 성찰 요약 생성 (목 데이터)
- 1개 다음 행동 제안 (목 데이터)
- 세션 완료 후 결과 페이지 이동

### 기술
- Next.js App Router
- TypeScript
- TailwindCSS
- React useState로 세션 상태 관리
- sessionStorage로 페이지 간 데이터 전달
- 목 데이터 (하드코딩된 질문/요약)

### 디자인
- 모바일 퍼스트 (375px 기준)
- 따뜻하고 차분한 색상
- 넉넉한 여백
- 깔끔한 타이포그래피
- 단일 CTA
