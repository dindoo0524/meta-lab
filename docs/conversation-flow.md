# Conversation Flow — 성장톡 세션

## 전체 흐름

```
[랜딩] → [세션 시작] → [질문 1] → [질문 2] → [질문 3] → [결과]
```

## 상세 단계

### Step 0: 랜딩 (`/`)
- "성장톡" 브랜딩 표시
- "오늘의 대화 시작하기" CTA 버튼
- 클릭 시 `/session`으로 이동

### Step 1: 세션 인트로 (`/session` — INTRO)
- 프롬프트: "오늘 하루 어떠셨나요?"
- 사용자가 자유롭게 입력 (textarea)
- "시작하기" 버튼으로 다음 단계

### Step 2: 질문 1 (`/session` — Q1)
- AI가 감정/에너지 관련 질문
- 예: "오늘 에너지를 가장 많이 쓴 건 뭐였나요?"
- 사용자 답변 후 "다음" 버튼

### Step 3: 질문 2 (`/session` — Q2)
- AI가 핵심 경험 관련 질문
- 예: "오늘 가장 오래 머물렀던 생각이나 감정은?"
- 사용자 답변 후 "다음" 버튼

### Step 4: 질문 3 (`/session` — Q3)
- AI가 변화/선택 관련 질문
- 예: "오늘을 다시 산다면 뭘 바꾸고 싶나요?"
- 사용자 답변 후 자동으로 결과 생성

### Step 5: 결과 (`/result`)
- 짧은 성찰 요약 (2~3문장)
- 내일의 한 가지 행동 제안
- "처음으로" 버튼 → 랜딩으로 복귀

## 기술 구현

### 상태 관리
- `SessionStep` enum: `INTRO → Q1 → Q2 → Q3 → COMPLETE`
- `use-session.ts` 커스텀 훅으로 관리
- React `useState` 사용 (글로벌 상태 없음)

### 페이지 간 데이터 전달
- 세션 완료 시 `sessionStorage`에 결과 저장
- `/result`에서 읽어와 표시
- 데이터 없으면 `/`로 리다이렉트

### 목 데이터 전략 (v0)
- 3개 질문 세트를 미리 정의
- 초기 입력의 키워드 기반으로 세트 선택 (단순 매칭)
- 요약과 다음 행동도 미리 정의된 템플릿에서 선택

## 질문 설계 원칙

- 열린 질문 (예/아니오가 아닌)
- 구체적 경험을 끌어내는 질문
- 감정 → 경험 → 변화 순서로 깊어지는 흐름
- 짧고 명확한 문장
- 설교나 유도 없음
